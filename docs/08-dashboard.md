# Dashboard

## Общая идея

Dashboard на `Streamlit` — это удобный человеческий вход в систему без необходимости вручную вызывать API или CLI.

## Что уже есть

### Обзор данных

Показывает:

- число варок;
- число размеченных варок;
- число новых варок;
- число пропусков;
- число доступных моделей;
- количество протоколов;
- количество повторных протоколов;
- распределение шагов по варкам.

### Прогноз варки

Позволяет:

- выбрать batch;
- выбрать контрольный этап;
- построить прогноз;
- увидеть интервалы, статус, confidence и top factors;
- посмотреть похожие варки;
- при необходимости показать компоненты и этапы.

### Модели

Показывает:

- число моделей;
- число champion;
- число challenger;
- список моделей и метаданные.

### Качество данных

Показывает:

- summary по качеству данных;
- проблемные поля;
- sample issues по выбранной batch.

## Как запускать

```bash
python -m app.cli run-dashboard
```
